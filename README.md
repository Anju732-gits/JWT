JWT Task Management API

A REST API built with **FastAPI**, **JWT Authentication**, and **Supabase** for secure task management.

## Features
- User Registration & Login
- JWT Authentication
- Password Hashing with bcrypt
- Category Management
- Task CRUD Operations
- Supabase Integration

## Tech Stack
- FastAPI
- Python
- Supabase
- JWT
- Pydantic

## Run

```bash
uvicorn main:app --reload
```
